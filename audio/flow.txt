For the **Toradex Verdin iMX8M Plus**, the build flow is very similar to other embedded Linux boards, but Toradex provides a complete BSP and OS called **Torizon OS** or **Yocto BSP**.

Here is the **complete build flow from scratch**, which most embedded Linux engineers follow.

---

# Overall Build Flow

```text
Host PC (Ubuntu 22.04)

    │

    ├── Install Build Tools

    │

    ├── Download Source Code

    │      ├── U-Boot
    │      ├── Linux Kernel
    │      ├── Device Tree
    │      └── RootFS / Yocto BSP

    │

    ├── Build Bootloader

    │      └── flash.bin

    │

    ├── Build Linux Kernel

    │      ├── Image
    │      ├── *.dtb
    │      └── Modules

    │

    ├── Build Root Filesystem

    │      ├── Busybox
    │      ├── Systemd
    │      ├── Qt
    │      └── Applications

    │

    ├── Create SD/eMMC Image

    │

    └── Flash to Board

            │

            Boot

            ROM

             ↓

            SPL

             ↓

           U-Boot

             ↓

        Linux Kernel

             ↓

        Device Tree

             ↓

          RootFS

             ↓

        User Application
```

---

# STEP 1 : Prepare Build Machine

Use:

```text
Ubuntu 22.04 LTS

RAM : 16 GB minimum

Storage : 150 GB free

CPU : 8 cores recommended
```

Install packages:

```bash
sudo apt update

sudo apt install -y \
git \
build-essential \
gcc \
g++ \
cmake \
wget \
curl \
python3 \
python3-pip \
gawk \
diffstat \
unzip \
texinfo \
chrpath \
socat \
libsdl1.2-dev \
xterm \
zstd \
lz4 \
u-boot-tools \
device-tree-compiler
```

---

# STEP 2 : Install Cross Compiler

Because:

```text
Host PC

x86_64

↓

Target Board

ARM64
```

You need:

```bash
sudo apt install gcc-aarch64-linux-gnu
```

Verify:

```bash
aarch64-linux-gnu-gcc --version
```

Example:

```text
aarch64-linux-gnu-gcc

gcc 12.x
```

---

# STEP 3 : Get Source Code

Normally source consists of:

```text
Source Tree

bootloader/

kernel/

device-tree/

rootfs/

applications/
```

---

## Bootloader Source

Clone:

```bash
git clone \
https://github.com/u-boot/u-boot.git

cd u-boot
```

---

## Kernel Source

Clone:

```bash
git clone \
https://github.com/torvalds/linux.git

cd linux
```

---

## Or Use Toradex BSP

Toradex maintains:

```text
meta-toradex

meta-freescale

poky

meta-openembedded
```

which are Yocto layers.

---

# STEP 4 : Build U-Boot

Move:

```bash
cd u-boot
```

Set environment:

```bash
export ARCH=arm64

export CROSS_COMPILE=aarch64-linux-gnu-
```

Configure:

```bash
make imx8mp_evk_defconfig
```

Build:

```bash
make -j8
```

Generated:

```text
u-boot.bin

SPL

flash.bin
```

---

# STEP 5 : Build Linux Kernel

Move:

```bash
cd linux
```

Set:

```bash
export ARCH=arm64

export CROSS_COMPILE=aarch64-linux-gnu-
```

Configure:

```bash
make defconfig
```

or

```bash
make menuconfig
```

Enable:

```text
Audio

ALSA

SAI

I2S

USB

Ethernet

PCIe

HDMI

DRM

GPU

Camera
```

Build:

```bash
make -j8
```

Output:

```text
arch/arm64/boot/Image

*.dtb

modules
```

---

# STEP 6 : Device Tree Build

Device Tree describes:

```text
GPIO

Audio Codec

Display

Ethernet PHY

USB

I2C Devices

SPI Devices
```

Example:

```dts
&i2c2 {

codec: wm8960@1a {

compatible = "wlf,wm8960";

reg = <0x1a>;

};

};
```

Compile:

```bash
dtc

-I dts

-O dtb

myboard.dts

-o myboard.dtb
```

Generated:

```text
myboard.dtb
```

# STEP 8 : Yocto Environment Setup

Clone:

```bash
mkdir yocto

cd yocto
```

Get:

```text
poky

meta-openembedded

meta-freescale

meta-toradex
```

Initialize:

```bash
source poky/oe-init-build-env
```

This creates:

```text
build/

conf/

tmp/

downloads/

sstate-cache/
```

---

## Configure Board

Edit:

```text
conf/local.conf
```

Set:

```bash
MACHINE = "verdin-imx8mp"
```

Example:

```bash
DISTRO = "tdx-xwayland"

MACHINE = "verdin-imx8mp"
```

# STEP 12 : Application Build

Example:

```text
my_project/

├── ui/

│

├── audio/

│

├── dsp/

│

├── ai/

│

├── drivers/

│

├── scripts/

│

└── CMakeLists.txt
```

Build:

```bash
mkdir build

cd build

cmake ..

make -j8
```

Generate:

```text
myapp
```

Copy:

```bash
scp myapp

root@board:/usr/bin
```

Run:

```bash
./myapp
```
