# 1. Set Up the Development Environment

On an Ubuntu PC:

```bash
sudo apt update

sudo apt install git build-essential \
cmake gcc g++ \
python3 python3-pip \
g++-aarch64-linux-gnu
```

Install useful tools:

```bash
sudo apt install \
u-boot-tools \
device-tree-compiler \
flex bison \
libssl-dev \
libncurses-dev
```

---

# 2. Build U-Boot Bootloader

Clone U-Boot:

```bash
git clone https://github.com/u-boot/u-boot.git

cd u-boot
```

Configure for i.MX8M Plus:

```bash
make imx8mp_evk_defconfig
```

Compile:

```bash
make CROSS_COMPILE=aarch64-linux-gnu- -j8
```

Output:

```text
u-boot.bin
spl.bin
flash.bin
```

These are the bootloader images.

---

# 3. Build Linux Kernel

Clone Linux:

```bash
git clone https://github.com/torvalds/linux.git

cd linux
```

Configure:

```bash
make ARCH=arm64 defconfig
```

or:

```bash
make ARCH=arm64 menuconfig
```

Enable:

```text
Sound support

I2S

S/PDIF
```

### Display

```text

GPU drivers
```

### Networking

```text
Ethernet

PCIe

WiFi
```

Compile:

```bash
make ARCH=arm64 \
CROSS_COMPILE=aarch64-linux-gnu- \
-j8
```

Outputs:

```text
Image

kernel modules
```

# 4. Device Tree

Device Tree describes the hardware.

Example:

```dts
sound {

compatible = "simple-audio-card";

simple-audio-card,name = "MyAudio";

simple-audio-card,cpu {

sound-dai = <&sai2>;

};

simple-audio-card,codec {

sound-dai = <&codec>;

};

};
```

Example GPIO:

```dts
gpio-led {

compatible = "gpio-leds";

status {

gpios = <&gpio1 5 GPIO_ACTIVE_HIGH>;

};

};
```

Compile:

```bash
dtc -I dts -O dtb myboard.dts -o myboard.dtb
```

# 9. DSP on Cortex-M7

The M7 core can run:

```text
FFT

EQ

Filters

Compressor

Reverb

Noise suppression
```

Simple FIR filter:

```c
float fir(

float input)

{

y =

b0*x[n]

+

b1*x[n-1]

+

b2*x[n-2];

return y;

}
```

Compile:

```bash
arm-none-eabi-gcc
```

This runs separately from Linux.

## Recommended order to learn

If you are a beginner, follow this sequence:

1. Learn **C and C++**.
2. Learn **embedded Linux basics**.
3. Build **U-Boot**.
4. Build the **Linux kernel**.
5. Learn **Device Tree**.
6. Build a **BusyBox root filesystem**.
7. Learn **ALSA audio programming**.
8. Build a **Qt GUI**.
9. Learn **Cortex-M7 bare-metal/FreeRTOS programming**.
10. Add **TensorFlow Lite** and NPU acceleration.

This is essentially the same software stack used in commercial products based on the Toradex Verdin iMX8M Plus, including industrial HMIs, smart speakers, digital musical instruments, and embedded DAWs.
